# <a name="option"></a>OPTION

Включает и отключает функции ассемблера.

## <a name="syntax"></a>Синтаксис

> **`OPTION`***Option-List*

## <a name="remarks"></a>Примечания

Доступны следующие варианты:

:::row:::
   :::column span="":::
      **`CASEMAP`**\
      **`DOTNAME`**\
      **`NODOTNAME`**\
      **`EMULATOR`**\
      **`NOEMULATOR`**\
      **`EPILOGUE`**\
      **`EXPR16`**
   :::column-end:::
   :::column span="":::
      **`EXPR32`**\
      **`LANGUAGE`**\
      **`LJMP`**\
      **`NOLJMP`**\
      **`M510`**\
      **`NOM510`**\
      **`NOKEYWORD`**
   :::column-end:::
   :::column span="":::
      **`NOSIGNEXTEND`**\
      **`OFFSET`**\
      **`OLDMACROS`**\
      **`NOOLDMACROS`**\
      **`OLDSTRUCTS`**\
      **`NOOLDSTRUCTS`**\
      **`PROC`**
   :::column-end:::
   :::column span="":::
      **`PROLOGUE`**\
      **`READONLY`**\
      **`NOREADONLY`**\
      **`SCOPED`**\
      **`NOSCOPED`**\
      **`SEGMENT`**\
      **`SETIF2`**
   :::column-end:::
:::row-end:::

Синтаксис языка — **`OPTION LANGUAGE:`** _`x`_ , где *`x`* — это один из,,, **`C`** **`SYSCALL`** **`STDCALL`** **`PASCAL`** , **`FORTRAN`** или **`BASIC`** . **`SYSCALL`**, **`PASCAL`** , **`FORTRAN`** и **`BASIC`** не поддерживаются с [`.MODEL`](dot-model.md) **`FLAT`** .
