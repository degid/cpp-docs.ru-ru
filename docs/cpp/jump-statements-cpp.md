# <a name="jump-statements-c"></a>Операторы перехода (C++)

Оператор перехода C++ выполняет немедленную локальную передачу контроля.

## <a name="syntax"></a>Синтаксис

```
break;
continue;
return [expression];
goto identifier;
```
