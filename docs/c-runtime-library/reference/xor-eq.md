# <a name="xor_eq"></a>xor_eq

Альтернатива оператору ^=.

## <a name="syntax"></a>Синтаксис

```C

#define xor_eq ^=
```

## <a name="remarks"></a>Примечания

Макрос создает оператор ^=.

## <a name="example"></a>Пример

```cpp
// iso646_xor_eq.cpp
// compile with: /EHsc
#include <iostream>
#include <iso646.h>

int main( )
{
   using namespace std;
   int a = 3, b = 2, result;

   result= a ^= b;
   cout << result << endl;

   a = 3;
   b = 2;

   result= a xor_eq b;
   cout << result << endl;
}
```

```Output
1
1
```

## <a name="requirements"></a>Требования
