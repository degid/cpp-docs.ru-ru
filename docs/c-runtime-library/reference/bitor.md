# <a name="bitor"></a>bitor

Альтернатива оператору &#124;.

## <a name="syntax"></a>Синтаксис

```C

#define bitor |
```

## <a name="remarks"></a>Примечания

Макрос создает оператор &#124;.

## <a name="example"></a>Пример

```cpp
// iso646_bitor.cpp
// compile with: /EHsc
#include <iostream>
#include <iso646.h>

int main( )
{
   using namespace std;
   int a = 1, b = 2, result;

   result = a | b;
   cout << result << endl;

   result= a bitor b;
   cout << result << endl;
}
```

```Output
3
3
```
